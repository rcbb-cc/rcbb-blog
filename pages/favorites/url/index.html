# 摸鱼

  • 996.icu (opens new window) 必看网站

  • DeepL (opens new window) 人工智能翻译，全世界最准的翻译

  • 简书 (opens new window) 有很多频道的创作社区

  • 开发热榜 (opens new window) 整合了多个开发者网站的热榜
