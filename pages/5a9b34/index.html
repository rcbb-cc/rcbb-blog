阿里云配置安全组

# 安全组入口

控制台 -> 产品与服务 -> 云服务器ECS -> 安全组 -> 点击安全组ID

云服务器ECS -> 安全组

点击安全组ID

入方向：我们提供出给外部访问的端口。

出方向：一般不限制，安全组出方向默认允许所有访问，即从安全组内ECS访问外部都是放行的。

# 配置入方向

例如之前的文章，我们修改 MySQL 的端口为 5555 了，现在我们开放 5555 的端口。

# 测试

使用 telnet 命令：telnet ip 5555

如果你的这个端口也是数据库端口的话，就会报密码错误。也证实了我们端口已经开放。

未开通的效果，无法打开到主机连接。
